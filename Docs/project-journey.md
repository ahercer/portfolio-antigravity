# Project Journey: Portfolio with Antigravity
**Owner:** Anna Hercer, Strategic Lead BI Engineer
**Date:** April 2026

## 1. Project Vision & Goal
The objective was to transform a professional CV into a high-impact, interactive portfolio that reflects 20+ years of expertise in the Microsoft Data Stack. The theme chosen was **"The Data Cosmos"**—a dark, sophisticated aesthetic with subtle "Antigravity" interactive elements.

## 2. Technical Stack
- **Framework:** React with TypeScript (via Vite) – Chosen for its speed and type safety.
- **Styling:** Vanilla CSS – Selected for full control over animations and performance.
- **Hosting:** Azure Static Web Apps – Picked to align with Anna's expertise in the Azure ecosystem.
- **CI/CD:** GitHub Actions – To automate the deployment process.

## 3. Key Milestones & Evolution

### Phase 1: Foundation & Design
- **Scaffolding:** Initialized the project using Vite. Cleaned up boilerplate to start with a "blank canvas."
- **Visual Identity:** Defined a CSS variable system based on Microsoft Fabric (Blue), Azure (Purple), and Power BI (Yellow).
- **Antigravity Effect:** Implemented a global utility class for smooth hover transitions (`translateY` and glowing shadows).

### Phase 2: Content & Context
- **Lead BI Identity:** Refined the professional summary to focus on *Analytics Engineering*, Kimball Methodology, and Dimensional Modelling rather than just Data Engineering.
- **Multilingual Support:** Built a custom React state-based translation system (EN/PL) to cater to both local and international markets.

### Phase 3: Interactive Features
- **Smart Contact Info:**
    - **Interactive Pin:** Linked to Google Maps (Mansfield, UK) with a color-shift hover effect.
    - **Active Email:** Implemented a `mailto:` link using a custom, high-visibility SVG icon.
- **LinkedIn Integration:** Added a professional SVG icon in the footer that directs users to Anna's profile.
- **CV Access:** Integrated a direct link to the full PDF on Google Drive, ensuring proper sharing permissions.

### Phase 4: Refinement & "Lessons Learned"
- **The "Ask Before Build" Rule:** After a few speculative attempts (SQL Wave background, Strategic Impact section) that didn't align with the vision, we established a strict "Propose & Approve" workflow.
- **Rollback Strategy:** We learned that "less is more." Keeping the design clean and focused on the CV data proved more effective than adding heavy background animations.

## 4. Deployment Guide (The Azure Step-by-Step)
For future reference, here is how we "spun up" the live version:
1. **GitHub Sync:** Pushed the local code to a new repo: `ahercer/portfolio-antigravity`.
2. **Azure Setup:** 
   - Created an **Azure Static Web App** (Free Tier).
   - Connected it to the GitHub repository.
   - Selected the **React** build preset.
   - Set the **Output location** to `dist` (Vite's default output folder).
3. **CI/CD:** Azure automatically created a GitHub Action that builds and deploys the site on every push.

## 5. Summary of Engineering Principles Used
- **Surgical Updates:** Using `replace` and `write_file` tools to keep the code clean.
- **Performance First:** Using CSS `transform` for animations to utilize GPU acceleration.
- **UX (User Experience):** Ensuring all links open in new tabs (`target="_blank"`) and provide visual feedback on hover.
