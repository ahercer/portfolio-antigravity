# Project Overview: Portfolio with Antigravity

Welcome to the official documentation for the **Anna Hercer Portfolio** project, also known as "The Data Cosmos."

## Vision
To create a high-impact, professional, and interactive portfolio that reflects 20+ years of expertise in the **Microsoft Data Stack**. The project uses a modern "Dark Mode" aesthetic with subtle "Antigravity" interactive elements.

## Key Features
- **Multilingual Support**: Full translation between English and Polish.
- **Responsive Design**: Optimized for mobile, tablet, and desktop.
- **"Antigravity" UI**: Custom hover effects providing a sense of depth and interactivity.
- **Direct CV Access**: Integrated PDF download and LinkedIn connectivity.
- **Azure-Native**: Built and deployed using Azure Static Web Apps.
